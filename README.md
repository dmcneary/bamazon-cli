# bamazon
  
### A simple CLI for "purchasing" products from my bike shop!
  
##### TECHNOLOGIES USED:
* Javascript
* Node.js
* npm
* MySQL

First, we are greeted by some lovely ASCII art and a list of present inventory, stored on a locally running MySQL server:
